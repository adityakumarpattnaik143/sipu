# Python Zero-Level Exam

Welcome to your exam! Below are three logic-building questions.

**Instructions:**
Do not write the testing logic or input data yourself. Open `exam_tasks.py` and fill in the logic inside the provided function templates. When you are ready to test, run `pytest EXAM/` in your terminal.

## Task 1: Reverse a String

**Description:**
Write a function `reverse_string(text)` that takes a string as input and returns the string in reverse order.

**Example:**
- `reverse_string("hello")` should return `"olleh"`
- `reverse_string("Python")` should return `"nohtyP"`

---

## Task 2: Count the Vowels

**Description:**
Write a function `count_vowels(word)` that takes a string and returns the number of vowels (`a, e, i, o, u`) present in the string. Ignore case sensitivity (both uppercase and lowercase vowels should be counted).

**Example:**
- `count_vowels("apple")` should return `2` (a, e)
- `count_vowels("EDUCATION")` should return `5` (E, U, A, I, O)

---

## Task 3: Calculate the Average

**Description:**
Write a function `calculate_average(numbers)` that takes a list of numerical values and returns their mathematical average. If the list is empty, the function should return `0`.

**Example:**
- `calculate_average([10, 20, 30])` should return `20.0`
- `calculate_average([1, 2, 3, 4, 5])` should return `3.0`
- `calculate_average([])` should return `0`
